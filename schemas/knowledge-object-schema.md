# Knowledge Object Schema

Required fields:

- `id`
- `version`
- `status`
- `priority`
- `title`
- `summary`
- `category`
- `intent`
- `audience`
- `answer_style`
- `confidence`
- `owner`
- `review`
- `tags`
- `search_terms`
- `scripture`
- `resources`
- `next_steps`
- `related`
- `last_updated`

Registry references use dot notation, such as `church_center.baptism`.
