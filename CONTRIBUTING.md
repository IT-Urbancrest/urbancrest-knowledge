# Contributing

## Before Adding Content

1. Search for an existing article that answers the same question.
2. Confirm the answer does not duplicate another document.
3. Use the knowledge object template.
4. Reference registry IDs for links, ministries, staff, and locations.
5. Add the document to `manifest.yaml`.
6. Add or update intent routing and tests.

## Review

Belief and doctrine articles require pastoral review. Operational details require factual review by the responsible ministry.
