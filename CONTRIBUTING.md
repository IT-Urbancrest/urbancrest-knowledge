# Contributing

## Workflow

1. Create or edit one article at a time.
2. Use the article template.
3. Keep each file focused on one user question.
4. Add search terms that reflect how a real visitor might ask the question.
5. Add related articles for natural follow-up.
6. Commit with a clear message.

## Commit Message Examples

```text
Add salvation FAQ article
Update baptism next step link
Improve visitor parking article
```

## Review Checklist

- Is the answer accurate?
- Is it warm and clear?
- Does it include a next step?
- Does it avoid unnecessary jargon?
- Are the tags and search terms helpful?
